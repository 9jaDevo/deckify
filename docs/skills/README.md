# Deckify Skills Folder

This folder defines implementation skills/capabilities for the Deckify MVP.

## Planned skill modules

1. `document-ingestion.md` - handle DOCX upload and raw text input.
2. `ai-provider-routing.md` - choose and call OpenAI or Grok.
3. `deck-generation.md` - transform AI output into slide-ready structure.
4. `pdf-export.md` - render generated content into downloadable PDF.
5. `history-tracking.md` - persist user generation history.

Each skill document should include:
- Goal
- Inputs
- Outputs
- Validation rules
- Failure handling
- Test checklist
